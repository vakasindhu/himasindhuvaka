My Projects

Implementation, Analysis and Detection of Phishing URLs

Category: Machine Learning

I developed a system to detect malicious websites by analyzing their URLs. The goal was to help users avoid phishing attacks without relying solely on browser warnings. Using Python and Scikit-Learn, I trained a machine learning model to recognize subtle patterns in URLs that indicate potential threats.

The model can classify websites in real-time and reached an accuracy of 92%, demonstrating the effectiveness of combining feature engineering with supervised learning. This project strengthened my understanding of cybersecurity challenges and practical machine learning deployment.

Object and Digit Classification on Android Devices

Category: Machine Learning / Mobile Development

I built an Android app that identifies objects and handwritten digits in real-time using TensorFlow Lite. I designed and trained convolutional neural networks optimized for mobile devices to enable fast and accurate recognition.

The app achieves low-latency predictions with 98% accuracy, making it suitable for real-world mobile applications. This project deepened my knowledge of deep learning deployment on resource-constrained devices.

Social Distance Detection and Alert System

Category: Deep Learning / Computer Vision

In response to public health safety needs, I developed a Python system that monitors social distancing in real-time using camera feeds. Leveraging TensorFlow and OpenCV, the system detects people and measures distances between them, triggering alerts when social distancing is violated.

By optimizing the algorithm, I reduced alert latency by 40%, enabling timely notifications in crowds of 50+ people. This project taught me how to balance speed and accuracy in real-time computer vision applications.

Research & Publications

Predict Work-Life Balance-Based Employee Attrition for Women

This study forecasts work-life balance using Random Forest, Gradient Boosting, and Support Vector Machines to help individuals, particularly women, manage their professional and personal responsibilities more effectively.

Random Forest excels in identifying complicated relationships by leveraging varied data sources such as employment positions and personal commitments. Gradient Boosting reliably recognizes detailed patterns, while SVM provides flexibility in decision-making parameters.

These algorithms provide individuals with vital insights into their work-life balance dynamics, allowing them to make informed decisions to improve their overall well-being.

Analysis of Liver Patient Dataset

This study uses data mining approaches to predict liver diseases based on patient data, aiming to improve diagnostic and prognostic accuracy.

With the rising incidence of obesity and poor lifestyles contributing to liver-related illnesses, data mining is critical for unearthing hidden information and making informed medical decisions.

By analyzing large volumes of healthcare data, administrators can improve service quality and patient outcomes through informed decision-making.

Fire Detection and Alerting System

Fire detection systems are critical for early discovery of fires, facilitating safe evacuation, and reducing property loss. This study presents a low-cost, easily deployable IoT-based solution for timely fire notifications.

The proposed system uses sensors that communicate data to a cloud platform, prompting warnings to fire departments depending on specified criteria, thereby improving response efforts.
